# The Happiness of Unsubscription
> "The absolute freedom is found not in adding, but in unsubscribing."

A minimalist web-based meditation logic inspired by the **Mucalinda Sutta**. This project represents the journey of the mind through four phases of liberation, translated into a modern digital context.

## 4 Phases of Liberation
1. **Viveka**: Silencing external noise.
2. **Abyapajjha**: Releasing all ill-will and conflict.
3. **Viragatta**: Unsubscribing from desire (The Minimalist Step).
4. **Asmimanassa Vinaya**: The dissolution of self (The Absolute Freedom).

## Tech Stack
- Vanilla JavaScript
- CSS3 (Minimalist High-Contrast UI)
- HTML5

## How to Run
Simply open `index.html` in any modern web browser to experience the loop of tranquility.
